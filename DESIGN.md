## 一句話

**俯視角彈幕戰術 roguelite**：撐過十波夜襲直到黎明 —— 打倒的敵人不會死，
會變成蒼白的「擊倒體」，走近按 G 把它收編成隊友（隊友＝你的命），或長按 G 吞噬它換取力量。

## 核心迴圈（每一次擊殺都問你一個問題）

```
擊殺 → 敵人進入擊倒狀態（25s 蒼白脈動的機會）
     → 收編（G）      隊友 = 額外生命（死亡自動接管隊友軀體）
     → 吞噬（長按 G）  職業化報酬：回血 / 武器入庫 / 能量
     → 武器掉落        撿槍 = 補給（無手動換彈匣經濟）
     → 連殺狂熱        4 秒窗口的速度/射速增益，停手就消退
波次清空 → 三選一升級卡（roguelite）→ 下一波更兇
第 10 波 = HOLLOW-31 頭目 → 黎明 = 勝利（可續戰長夜）
```

## 從原作原封保留的（已被驗證的手感）

- **槍管模型**：follow/sway/recoil 三個純量 = 每把槍的物理個性（6 把武器數值表原封）
- **公平三件套**：敵傷 ×0.6、單發上限 55%、敵彈射程 ≤ 你的武器射程
- **擊殺回饋疊層**：每發微震 → 白熱槍口焰 → 3-tick 0.18× hit-stop（僅自己的擊殺）→ 出生定色粒子爆炸（15/45/25/15 火焰調色盤）→ 連殺橫幅（8 階顏色+音高爬升）
- **84Hz 模擬時鐘**（原作隱藏的 1.4× timescale，所有 tick 常數原封移植）
- **構成主義調色盤**：`#C8261C` 紅 / `#E8E4D8` 米 / `#1A1A1A` 黑 / `#0F0F0F` 地板（永不變亮）+ 4 組時辰色盤（只動灰階）
- **視野即戰場**：140° 視錐 + 省略式戰爭迷霧（3s 殘影記憶）+ 音源警示（開槍 = 廣播你的位置）
- **敵人 FSM**：掩體點（每障礙 4 點 32u）、60/60 偷看節拍、最遠者側翼包抄、30% 血潰逃、命中回溯漣漪 —— 純規則，無神經網路
- **5 層合成槍聲**（click/crack/lowpass 距離線索/低頻/延遲回聲）、心跳血條、氛圍低音墊 —— 全 WebAudio，零 mp3
- **一鍵重生節拍**：死亡 → 擊殺回放線 + KILLED BY → SPACE 接管隊友或結束
