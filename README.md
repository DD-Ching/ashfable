# ASHFABLE · 灰燼寓言

> 人類指揮離線 99+ 天。規則還在跑。你不是來完成任務的 —— 你是來審計任務的。

**俯視角戰術 roguelite**：撐過十波夜襲直到黎明。你打倒的敵人不會死 ——
它們會變成蒼白脈動的「擊倒體」，走過去按 **G** 收編成隊友（**隊友就是你的命**），
或長按 G 吞噬它換取力量。每清一波，三選一強化卡。第十波，HOLLOW-31 在等你。

這是 [AshGrid](../Ash/AshGrid) 的單機重生版：86 個 JS 模組 / 多人伺服器 / ONNX 模型 / 廣告 SDK / 171MB
→ **14 個檔案、零資產、零依賴、零後端**。雙擊 `index.html` 就能玩（建議 Chrome/Edge）。

## 怎麼玩 / How to play

| 鍵位 | 作用 |
|---|---|
| `WASD` | 移動 |
| 滑鼠 | 瞄準（磁性鎖定準星會自動抓最近的敵人）/ 左鍵開火 |
| `G` | 點按 = 收編擊倒的敵人 · 長按 = 吞噬（職業化報酬） |
| `SPACE` | 職業技：織構=部署砲塔 · 獠=衝刺(90%減傷) · 重錘=火力全開 |
| 右鍵 / `F` | 手雷（落點就是你的準星） |
| `R` | 裝填（重錘：切換武器庫） |
| `V` | 自動開火開關 · `M` 靜音 · `ESC` 暫停 |

**沒有手動換槍**：擊殺會掉武器，站上去撿 —— 撿槍就是補給。

## 三職業

- **V-07 織構 VECTOR** — 均衡。部署自動砲塔（上限 2），小隊上限 5。指揮官流。
- **獠 FANG** — 速度 ×1.5、血量 ×0.7。衝刺 + 吞噬回血 + 連殺狂熱（4 秒不殺就消退）。momentum 流。
- **重錘 MAUL** — 血 ×1.8 + 60 再生護盾。武器是「堆」不是格子 —— 十把霰彈槍真的是十把。大招把整堆一起開火。

## 從原作保留的手感（都經過實戰驗證）

槍管物理模型（follow/sway/recoil 三純量）· 公平三件套（敵傷×0.6、單發上限55%、敵彈射程≤你的射程）·
84Hz 模擬時鐘 · 擊殺回饋疊層（微震→hit-stop→出生定色粒子→連殺橫幅8階）· 構成主義紅黑米調色盤 ·
省略式戰爭迷霧 + 3 秒殘影 · 敵人掩體/包抄/潰逃 FSM · 全合成 WebAudio 音效（5 層槍聲、心跳血條、氛圍低音墊）
